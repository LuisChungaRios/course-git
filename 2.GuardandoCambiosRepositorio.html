Iniciar git

  • git init = iniciar git
  • git status = ver el estado del repositorio
  • git add archivo = a√±ade el archivo al repositorio (q para salir)
  • git commit -m "descripcion de lo que estas haciendo" = confirmas los cambios en git
  • git log = ver el historial de los commit confirmados
  • git diff = muestra los cambios que se hicieron al archivo diferencia del ultimo estado del archivo respecto a los archivos que ya estan confirmados
  • git diff --staged = las diferencias de los archivos (q primero agregaste al repositorio pero no confirmaste e hiciste algunos cambios )
  • git add . = agregar todos los archivos a la zona de preparacion
  • git reset HEAD archivo == sacar el archivo de la zona de preparacion
  • git add -A = agrega archivos que estamos siguiendo
  • git commit --amend == cambia la descripcion del archivo confirmado en el repositorio (es decir si te equivocaste en la descripcion y por defecto se abre el editor de texto que configuraste )
  • .gitignore == ignorar los archivos para que git no lo tome en cuenta
