• git commit --amend -> se utliza para modificar un commit realizado, es decir si realize un commit a un archivo y se me olvido agregar otro archivo, realizo el commit amend para unir los dos archivos como un solo cambio, por defecto se abrira el editor de texto modificado en el global
  • git log --decorate --oneline --all --> ver el historial en forma ordenada
  • git reset HEAD 'file'-> sacar el archivo del area de preparacion
  • git checkout -- 'archivo' -> eliminar las modificaciones que se iso al archivo, vuele a su version anterior
  • git clone 'urlrepositoriogithub' -> clona el repositorio por defecto la carpeta se llamara igual que el repositorio
  • git clone 'urlrepositorioguthub' 'nombredecarpeta'-> clona el repositorio y la das el nombre a la carpeta
