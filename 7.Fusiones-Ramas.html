FUSIONES DE RAMAS

  • git -merge nombre_rama -> Incorpora otra rama, en la rama actual
  • git branch -v -> nos muestra las ramas con la descripcion del ultimo commit
  • git branch -d nombre_rama -> elimina una rama,solo cuando esa rama ya a sido fusionada, de lo contrario, no nos dejara eliminar
  • git branch -D nombre_rama-> fuerza la eliminacion de la rama, este o no, fusionada
  • git branch --no-merged -> ver que ramas no se an fusionado, a la rama actual
  • git branch --merged -> ver las ramas que an sido fusionadas
  • git checkout -b nombre_rama -> ir a una nueva rama que vamos a crear
