Etiqueta Ligera

  • git tag 'nombre de la etiqueta' -> Se le asigna al ultimo comiit que se iso

    • Etiqueta Anotada

    Se guardan en la base de datos de Git como objetos enteros.Tienen un checksum; contienen el nombre del etiquetador, correo electronico y fecha, y tienen un mensaje asociado

  • git tag -a nombre_etiqueta -m '-m para el mensaje puedes hacer varios parrafos'-> etiqueta anotada
  • git show nombredelaetiqueta -> muestra la etiqueta, los cambios que se hicieron y el commit

    • Etiquetas para un commit especifico

  • git tag etiqueta sumadecomprobacion -> agrega una etiqueta en un commit especifico
  • git tag -> lista las etiquetas en orden alfabetico
  • git tag -l "filtro de las etiquetas"-> filtrar las etiquetas
