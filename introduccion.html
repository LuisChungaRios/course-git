COMANDOS PARA CONFIGURAR GIT

  • git config --global user.name "Luis chunga"
  • git config --global user.email "luischungar7@gmail.com"
  • git config --global core.editor subl =editor para git
  • git config --list == muesta las configuraciones que se iso
  • git config --user == muesta las configuraciones que se iso en el usuario
  • git config --editor == muesta las configuraciones que se iso en el editor

Comandos de ayuda en git

  • git config --help
  • git help config
