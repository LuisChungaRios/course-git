• git commit == la descripcion la agregaremos desde el editor que configuramos
  • git commit -m "descripcion si cerrar las comillas == te agrega un salto de linea
  • git commit -a -m ==saltar la zona de preparacion de los archivos que estamos siguiendo(-a == los archivos que estamos rastreando)
  • git rm ==elimina archivos del repositorio
  • git checkout -- archivo == recupera los cambios hechos(eliminados desde el editor)
  • git mv archivoInicial archivoRenombrado ==renombrar el archivo
  • git log --oneline == historial abreviado
  • git log --graph == historial en grafico
  • git log --oneline --graph ==historial mezclado
  • git log -2 == historial de los dos ultimos commit que hicimos
  • git log --help == muestra la ayuda
  • git log --pretty=format:"%h - %an, %ar : %s" ==historial del log con algunas especificaciones
  • git log --after="2016-04-2018" == muestra el historial de la fecha indicada
  • git log --before="2016-04-2018 00:00:00" == muestra el historial de la fecha indicada (se puede mezclar el after y before)
