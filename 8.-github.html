Pasos

Primero creamos nuestra cuenta en github

Creamos el repositorio

Copiamos los codigos del repositorio

Por ejemplo

git remote add origin https://github.com/LuisChungaRios/course-git.git git push -u origin master

  • Subir los commit al repositorio de github -> git push origin master
  • Para llevar el repositorio a nuestra cuenta u otra cuenta damos click en Fork
  • Luego clonamos

    • Para crear un aporte a ese repositorio creamos una rama y en esa rama creamos todo nuestro aporte

    subimos nuestro aporte -> git push origin nombre_dela_rama

    una vez que subimos,en github selecionamos nuestra rama (branch) y seleccionamos pull request o compare pull request

    Luego nos aparecera un panel de opciones

    Si queremos confirmar el aporte en la otra cuenta , hacemos un merge pull request y se nos abrira un panel de opciones para agregar el titulo y descripcion

  • git fetch origin master -> jalar los cambios de github(when se ase un fork)
  • git merge origin/master-> unir los cambios de github
